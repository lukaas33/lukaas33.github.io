Boarding


My Android boarding screen.


  • Story

    I got an email from materialUp, a design website, about a contest. The assignment was to make an onboarding screen, which is a screen you see when using an app for the first time.
    I used html to make a concept of a boarding screen for Android.

    Although I didn't win the contest, I learned a lot about html, using reusable code and time management.

  • Story

    I made something I wanted to make for a while; A flat picture generator.

  • Story

    This website was based on an earlier project. Some time ago I made some space based wallpapers for my wallpapers collection.

    I learned how to make animations in the html canvas using paper.js.
    I also used a lot of object oriented programming. I divided my objects in lots of different classes that all have different properties.
